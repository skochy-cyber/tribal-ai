Security

Your security is our top priority. Here's how we protect you.

🛡️All data encrypted in transit and at rest

Authentication

  • JWT-based secure authentication
  • Passwords hashed with bcrypt (12 rounds)
  • Two-factor authentication (TOTP) available
  • Google OAuth social login
  • Session management — see and revoke all devices

Data Protection

  • All API traffic encrypted via TLS/HTTPS
  • Chat data encrypted at rest in MongoDB
  • No data shared with third parties
  • No data used for AI model training
  • GDPR compliant — full data export available

Infrastructure

  • Rate limiting on all API endpoints
  • DDoS protection via Cloudflare
  • Automatic backups
  • Isolated sandbox for code execution
  • Input validation on all endpoints

Access Control

  • Role-based access (User, Admin)
  • Admin actions fully logged
  • API keys scoped to Pro plan only
  • Team-based permissions

Incident Response

If you discover a security vulnerability, please report it responsibly to obasanjosamuel404@gmail.com. We respond within 24 hours.
