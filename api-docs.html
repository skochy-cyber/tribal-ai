API Documentation

Integrate Tribal AI into your apps. Access chat, models, templates, and analytics via our REST API.

Overview

The Tribal AI API uses REST endpoints over HTTPS. All requests require an API key passed via the Authorization header. Responses are JSON.

Base URL: https://api.tribalai.dev/v1
Authentication

Include your API key in the Authorization header as a Bearer token.

Request Header
Authorization: Bearer tribal_sk_your_api_key_here

Chat Completion
POST /api/chat
Send a message and receive an AI response. Optionally specify a chat ID to continue an existing conversation.
ParameterTypeRequiredDescription
messagestringRequiredThe user message
chatIdstringOptionalContinue an existing chat
modelstringOptionalOverride default model
Example
curl -X POST https://api.tribalai.dev/v1/chat \ -H "Authorization: Bearer tribal_sk_xxx" \ -H "Content-Type: application/json" \ -d '{"message": "Explain quantum computing", "model": "claude-sonnet-4"}'
List Chats
GET /api/chats
Retrieve all chat conversations for the authenticated user.
Response
[ { "_id": "chat_abc123", "title": "Quantum Computing Basics", "createdAt": "2026-07-21T10:30:00Z", "messageCount": 12 } ]
List Models
GET /api/models
Get available AI models and the user's current selection.
File Upload
POST /api/upload
Upload files (images, PDFs, code) for AI analysis. Max 10MB per file.
ParameterTypeRequiredDescription
messagestringOptionalContext message for the upload
filesarrayRequiredArray of file objects with name, type, b64
